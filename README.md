## 使用说明

1. UI 编辑器：

使用 fairyGUI：https://www.fairygui.com/

内置插件说明：https://github.com/daichangxin/gen-package-defines-plugin

2. 开发：

```
npm run dev
```

3. 打包：

```
npm run build
```
